# Setting up the project

This tutorial will go through the steps on how to setup the WeaverCore development project for creating a new scene

## Step 1 : Download the project files

### Download the project files from [here](https://www.dropbox.com/s/q3peusxn97zf8pz/WeaverCore%20Dev%20Project.zip?dl=1). This will give you a zip file with the entire project inside of it

![Zip Downloaded](https://user-images.githubusercontent.com/12601671/141728366-8f477034-aedc-4d42-99b5-ee72b4de8fd8.PNG)

### Right click on the downloaded zip and select "Extract All"

![Zip Extract All](https://user-images.githubusercontent.com/12601671/141728447-cd9974cd-1802-463f-9256-e2217566828e.PNG)

### Select a place to extract the files

![Zip Extract Files](https://user-images.githubusercontent.com/12601671/141728505-409bcf0f-5721-4605-be5d-a52c68af8939.PNG)

### Once extracted, you should have a folder with all the project files inside of it

![Dev Project Extracted](https://user-images.githubusercontent.com/12601671/141728601-ebcde38f-5c6c-4374-a040-334a8229a3d9.PNG)


### With the files extracted, open up the Unity Hub, go to the "Projects" tab, and click the "Add" button

![Unity Hub Add Project](https://user-images.githubusercontent.com/12601671/141728814-dc4535be-6afb-4531-bc7f-3a93e94de151.PNG)

### Select the folder that contains all the project files and click on "Select Folder"

![Unity Hub Add Project Window](https://user-images.githubusercontent.com/12601671/141729028-b3514aa3-0a01-47f1-8b32-ef2480883ce5.PNG)


### With the project added to the Unity Hub, click on the project (circled part) and this will load up the project into the Unity Editor (This may take a while : approximately 15 minutes)

![Unity Hub Project Added](https://user-images.githubusercontent.com/12601671/141729159-a66c1cdb-9f43-405f-8b6b-5bb094a265fb.PNG)


### Congrats! You have loaded the project! Note: if it prompts you updated to a new version, click on "Skip New Version"

![Project Loaded](https://user-images.githubusercontent.com/12601671/141730305-0c6704e3-353a-4e27-9373-0a49dfab79d7.PNG)
